# Boxing Stopwatch (Секундомер для бокса)

Простой консольный таймер раундов на Python. Подходит для отработки раундов с
паузами на отдых.

## Установка
Python 3.11+ уже достаточно. Дополнительные зависимости не нужны.

## Запуск
```bash
python boxing_timer.py --rounds 3 --round-seconds 180 --rest-seconds 60
```

Доступные параметры:
- `--rounds` — количество раундов (по умолчанию 3).
- `--round-seconds` — длительность раунда в секундах (по умолчанию 180, т.е. 3 минуты).
- `--rest-seconds` — отдых между раундами в секундах (по умолчанию 60).
- `--warmup-seconds` — необязательная разминка перед первым раундом (по умолчанию 10).
- `--no-bell` — отключить звуковой сигнал в конце интервала.

Остановить таймер можно сочетанием `Ctrl+C`.
